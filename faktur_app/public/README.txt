Faktur App

Aplikasi Faktur Penjualan Berbasis Web menggunakan HTML, PHP, CSS, JS (jQuery), dan MySQL.  
Dibuat untuk keperluan Uji Kopetensi Pemograman.

Fitur:
- Login & Logout
- CRUD Perusahaan, Customer, dan Produk
- Input & Cetak Faktur Penjualan
- Hitung otomatis (Subtotal, PPN 11%, DP 50%)
- Stok produk otomatis berkurang saat transaksi

Teknologi:
- PHP 8 (Laragon)
- MySQL
- Bootstrap 5
- jQuery
